1. Creating requirement.txt 
pip freeze >requirements.txt

2. Creating the virtual environment 
python -m venv recommendation_system_env

3. Activate the virtual env
.\recommendation_system_env\Scripts\activate

4. Installing the requirement.txt
pip install -r requirements.txt

5. After activating virtual env, run all cells

NOTE- 
1- If kernel problem occurs, then install desired  kernel in virtual env 
2- Always stay in recommendation_system_env kernel
